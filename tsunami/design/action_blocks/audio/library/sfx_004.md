# sfx_004 — `jump`

**Archetype:** movement (player jump, hop, bounce)
**Use cases:** platformer jump, pogo-hop, spring-launch
**Variants:** `jump_high` (double-jump), `jump_small` (hop)

## Description

**Upward** pitch sweep, square wave, 25% duty cycle for the Mario-ish
thin square tone. `p_freq_ramp: +0.4` climbs; short sustain and
decay keep it from overstaying. No punch — this is "light and springy."

## Variants

**`jump_high`** (bigger, more air):
```json
{ /* as above */, "p_base_freq": 0.3, "p_freq_ramp": 0.55,
  "p_env_sustain": 0.15, "p_env_decay": 0.35 }
```

**`jump_small`** (a hop, low effort):
```json
{ /* as above */, "p_base_freq": 0.4, "p_freq_ramp": 0.25,
  "p_env_sustain": 0.06, "p_env_decay": 0.15 }
```

Short sustain + low ramp = a bunny-hop feel.
