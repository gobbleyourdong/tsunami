# sfx_020 — `step`

**Archetype:** footstep / minor contact / continuous-movement feedback
**Shape:** canonical `SfxrParams`.
**Use cases:** player footstep (repeating during walking), light-
contact-with-surface, typewriter-key (UI), meter-tick (UI)
**Variants:** `step_grass`, `step_stone`, `step_metal`

**Note:** `step` is typically played at a regular rate (e.g. every
~400ms during movement). Could be `play_sfx_ref` on a timer, OR an
engine-level footstep system could trigger it on leg-plant animation
events. Implementation choice for the implementing instance.

## Description

Very short (~80ms total) low-noise burst. Punchy attack, fast decay,
low baseFreq + LPF at 0.4 = "soft thump." Low volume (0.25) — this
plays REPEATEDLY during any movement, so must be non-intrusive.

## Variants

**`step_grass`** (soft grass / dirt):
```json
{ "waveType": "noise", "envelopePunch": 0.25, "envelopeDecay": 0.08,
  "baseFreq": 0.28, "lpFilterCutoff": 0.3, "masterVolume": 0.2 }
```

Lower freq + stronger LPF = muffled.

**`step_stone`** (firm stone / tile / default):
```json
{ "waveType": "noise", "envelopePunch": 0.4, "envelopeDecay": 0.08,
  "baseFreq": 0.4, "lpFilterCutoff": 0.55,
  "lpFilterResonance": 0.1, "masterVolume": 0.3 }
```

**`step_metal`** (clang-adjacent, grated floors):
```json
{ "waveType": "noise", "envelopePunch": 0.5, "envelopeDecay": 0.12,
  "baseFreq": 0.55, "freqRamp": 0.05,
  "hpFilterCutoff": 0.2, "lpFilterResonance": 0.3,
  "lpFilterCutoff": 0.85, "masterVolume": 0.35 }
```

Higher freq + bright LPF + HPF + resonance = "metallic walkway."

## Implementation note

A surface-type system would be ideal:
- Archetype has `surface_tag: 'grass' | 'stone' | 'metal' | ...`
- When the player's `step_emit` condition fires (leg-plant event),
  compiler looks up the archetype's surface_tag → plays the right
  variant.

This is out of current sfxr scope (the surface-detection is gameplay
code), but the CONTENT (multiple step variants) is on my side.
Covered by having `step_grass`, `step_stone`, `step_metal` in the
library.
