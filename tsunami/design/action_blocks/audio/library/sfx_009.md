# sfx_009 — `break_glass`

**Archetype:** shatter / destructible / breakable
**Shape:** canonical `SfxrParams` per synthesis attempt_001 §3.
**Use cases:** breakable crate, window shatter, glass-cabinet
destruction, ice-break, jar-smash

## Description

High-pitched noise with strong punch and upward-then-downward freq
ramp (shards scattering then settling). `repeatSpeed: 0.4` creates
the "multiple pieces" retriggering effect. HPF at 0.3 strips low
rumble, keeping the bright shatter character. LPF ramp -0.4 closes
filter over decay, simulating the sound source diffusing away.

## Variants

**`break_wood`** (dull, fibrous):

Lower baseFreq + LPF at 0.5 removes the bright sparkle = "cracking
wood" feel.


Low freq + resonant LPF + flanger = big, heavy, stone-shatter.
