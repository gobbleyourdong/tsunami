# sfx_006 — `powerup`

**Archetype:** powerup / level-up / buff-gained
**Shape:** canonical `SfxrParams` per synthesis attempt_001 §3.
**Use cases:** Mario mushroom, Zelda heart container, Mega Man powerup
capsule, any "you just got stronger" moment
**Variants:** `powerup_minor` (small buff), `powerup_epic` (big event)

## Description

Rising pitch sweep + arpeggio. `arpMod: 0.3` + `arpSpeed: 0.4` gives
a stepped 2-3-note rising pattern. `freqRamp: 0.2` adds continuous
upward slide. Medium-length sustain (~300ms total). 50% duty cycle
for a full, confident square-wave timbre.
