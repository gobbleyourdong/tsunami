# sfx_003 — `explosion_small`

**Archetype:** destruction
**Use cases:** enemy pop, block break, firework, mine trigger
**Variants:** `explosion_large` (fuller), `explosion_muffled` (underwater/padded)

## Description

Noise burst with heavy punch envelope and downward LPF sweep.
wave_type 3 = noise. The LPF ramp (-0.3) closes the filter over the
decay, simulating air-absorption / muffling tail. `p_freq_ramp: -0.2`
drops the base noise pitch slightly, giving a sense of mass settling.

## Variants

**`explosion_large`** (deeper, longer, bass-heavy):
```json
{ "wave_type": 3, "p_env_punch": 0.9, "p_env_decay": 0.7,
  "p_base_freq": 0.18, "p_freq_ramp": -0.25,
  "p_lpf_freq": 0.85, "p_lpf_ramp": -0.25, "sound_vol": 0.8 }
```

**`explosion_muffled`** (underwater, silencer, "pad" surface):
```json
{ "wave_type": 3, "p_env_punch": 0.4, "p_env_decay": 0.5,
  "p_base_freq": 0.2, "p_freq_ramp": -0.1,
  "p_lpf_freq": 0.4, "p_lpf_ramp": -0.2, "p_hpf_freq": 0.1,
  "sound_vol": 0.4 }
```

Aggressive LPF at 0.4 removes high-frequency "bite" — the signature
of muffled explosions.
