# sfx_002 — `laser_small`

**Archetype:** projectile (player-fired small arm)
**Use cases:** pistol shot, small laser, bullet fire, shuriken throw
**Variants:** `laser_pulse` (rapid repeat), `laser_beam` (sustained)

## Description

Downward pitch sweep from mid-range to low, square wave, quick decay.
Classic "pew" of arcade shmups. `p_freq_ramp: -0.45` gives the
characteristic drop; `p_freq_limit: 0.15` stops the slide at a defined
floor so successive shots sound consistent.

## Variants

**`laser_pulse`** (rapid, automatic-fire feel):
```json
{ /* as above */, "p_repeat_speed": 0.5, "p_env_decay": 0.1 }
```

**`laser_beam`** (charged shot, sustained):
```json
{ /* as above */, "wave_type": 2,
  "p_env_sustain": 0.45, "p_env_decay": 0.3,
  "p_freq_ramp": -0.15, "p_vib_strength": 0.2, "p_vib_speed": 0.4 }
```

Sine wave + longer sustain + slight vibrato = sci-fi beam.
