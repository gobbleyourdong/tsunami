# sfx_011 — `death`

**Archetype:** player/enemy death sting (primary archetype)
**Shape:** canonical `SfxrParams`.
**Use cases:** player-killed, enemy-defeated (big), boss-defeated
**Variants:** `death_player` (longer, more emphasized), `death_enemy_small`

## Description

Long downward sweep (freqRamp: -0.5 with freqDeltaRamp: -0.1 for
accelerating drop). Slight vibrato for "wobbling fall" character.
Duty ramp opens the wave over time, adding a slow-morph feel.
Duration ~750ms. Feels like something losing its animating energy.

## Variants

**`death_player`** (longer, more emphasized — game-over candidate):
```json
{ "waveType": "square", "envelopeSustain": 0.35, "envelopeDecay": 0.9,
  "baseFreq": 0.5, "freqRamp": -0.55, "freqDeltaRamp": -0.15,
  "vibratoStrength": 0.2, "duty": 0.25, "lpFilterCutoffRamp": -0.3,
  "masterVolume": 0.65 }
```

**`death_enemy_small`** (shorter, part of the action rhythm):
```json
{ "waveType": "square", "envelopeSustain": 0.1, "envelopeDecay": 0.2,
  "baseFreq": 0.6, "freqRamp": -0.35,
  "duty": 0.5, "masterVolume": 0.4 }
```

Shorter sustain + smaller ramp = "popped" rather than "perished" —
good for wave-spawner mooks.
