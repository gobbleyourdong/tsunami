# sfx_015 — `charge`

**Archetype:** weapon charging / spell casting / meter filling
**Shape:** canonical `SfxrParams`.
**Use cases:** charge-shot buildup, spell-incantation hum, meter-fill
indicator, train-about-to-arrive, engine-starting
**Variants:** `charge_quick`, `charge_powerup` (charges to release)

**Note:** Ideally a `charge` SFX *loops* while the button is held,
then stops (or triggers a release SFX) when the button releases. This
uses the `play_sfx_loop` / `stop_sfx_loop` ActionRefs queued for
synthesis attempt_002.

## Description

Sawtooth wave with slow attack (0.1s) and long sustain (0.8). FreqRamp
upward (0.08) — pitch slowly rises while charging. Slight vibrato
(speed 0.25) adds a "flutter" to signify instability / energy
gathering. LPF opens slightly over sustain (ramp +0.15) — the sound
brightens as charge completes. Flanger adds the "energy-weaving"
harmonic shimmer typical of charge sounds.

## Variants

**`charge_quick`** (short charge — counter-attack / parry-window):
```json
{ "waveType": "sawtooth", "envelopeAttack": 0.03, "envelopeSustain": 0.2,

```

**`charge_powerup`** (full-power charge shot buildup to release):
```json
{ "waveType": "sawtooth", "envelopeAttack": 0.2, "envelopeSustain": 1.5,
  "envelopeDecay": 0.3, "baseFreq": 0.15, "freqRamp": 0.15,
  "vibratoStrength": 0.2, "vibratoSpeed": 0.4,
  "lpFilterCutoff": 0.5, "lpFilterCutoffRamp": 0.35,
  "flangerOffset": 0.3, "masterVolume": 0.55 }
```

Longer sustain + bigger flanger = "megabuster charging." Pair with a
`laser_beam` (sfx_002 variant) on release.
