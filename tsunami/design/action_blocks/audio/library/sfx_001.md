**Archetype:** pickup
**Use cases:** coin, ring, gem, score token, keyfind chime
**Variants:** `pickup_coin_v2` (deeper), `pickup_coin_metallic` (steel ring)

## Description

Two-note ascending pluck (arpeggio). Square wave with punchy envelope,
arp_mod 0.5 gives one-octave jump mid-sound. Instant attack, brief
sustain, quick decay. Feel: "clean, happy, confirming."

Sine wave + HPF opens it into a "shiny" register. Third variant below
0.1 on punch gets a muffled feel — use for collecting-in-mud contexts.
