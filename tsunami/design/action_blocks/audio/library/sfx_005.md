# sfx_005 — `hit_flesh`

**Archetype:** impact (soft)
**Use cases:** enemy-hit, player-damaged, melee-connect, grab
**Variants:** `hit_metal`, `hit_stone`, `hit_glass`

## Description

Short noise burst, medium-low base pitch, LPF at 0.55 removes the
"thin" high frequencies to give a softer "thunk." Punch envelope
starts sharp but decays in ~180ms — an impactful but not sustained
hit. Characteristic "oof" feel.
