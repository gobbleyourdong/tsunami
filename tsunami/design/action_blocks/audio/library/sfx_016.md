**Archetype:** milestone / advancement / character-progression
**Shape:** canonical `SfxrParams`.
**Use cases:** XP threshold hit, skill unlocked, rank promotion,
world-map node revealed, achievement-earned

## Description

Larger than `confirm` (sfx_014) — longer sustain, stronger punch,
higher arp interval (0.52 gives ~octave-plus, creating a triumphant
"major arp" feel). `freqRamp: 0.1` adds rising pitch for "growing
stronger" feel. Slight flanger for sparkle.

Duration ~800ms. Occupies the same functional slot as Mario's 1-UP
or Zelda's heart-container-get (game_013 lesson).
