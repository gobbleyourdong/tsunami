# sfx_013 — `missile`

**Archetype:** projectile (larger / physical / launched)
**Shape:** canonical `SfxrParams`.
**Use cases:** rocket-launch, missile-fire, torpedo, grenade-throw,
power-shot with mass
**Variants:** `missile_small`, `missile_homing` (with slight pitch
bend on launch)

## Description

Whoosh-launch with initial punch (launch ignition) and sustained
"traveling" noise. Mid-range baseFreq (0.35) with slight upward ramp
that tapers off (freqDeltaRamp: -0.1) simulates a missile
accelerating then stabilizing. Flanger gives a "jet stream" quality.
Soft HPF strips low rumble to keep it clean against music.


missiles / seekers / heat-trackers.
