What this is. Per-pixel raymarch into a flat primitive SDF buffer, evaluated on the GPU via a WGSL fragment shader. Camera is a free orbit pinned at fixed elevation, ortho-projected to the canvas.
Architecture parallel. Same primitive vocabulary as the engine's character3d raymarch renderer (sphere, box, capsule, cylinder, torus, smooth-union via blend groups). This is a standalone POC built fresh in TS+WGPU; no engine code was modified. The intent: future integration with the iso pixel cache so 3D shapes feed into the same destruction/particle pipeline.
